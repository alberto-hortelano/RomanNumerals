# RomanNumerals
Roman numeral library

## Start time:
 - 1517758140187

	1. install: `npm install`
	1. build: `npm build`
	1. test:
		* src: `npm run test_src`
		* build: `npm test`

## Usage;
	- `import { romanNumeral } from romannumerals;`
	- `var num = new romanNumeral('XVI')`
	- To integer: `num.toInt()`
	- To roman: `num.toString()`
